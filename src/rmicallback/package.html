This package contains a simple RMI client-server callback demo with instructions.

Instructions for running the simple RMI client-server callback demo

  1. Check out the demo project from CVS.
  2. Compile the project using ant initially and after each modification of the Java code.
  3. Assume you have chosen two different workstations, ws1 and ws2, preferably next to each other in a lab. (For testing purposes, ws1 and ws2 can be the same.) The server will run on ws1, and the client on ws2.
  4. Open a DOS window on each workstation. To find out the IP (Internet Protocol) Address of a workstation, type 
    ipconfig
    in a DOS window. Let ip1 be the IP Addresses of ws1.
  5. Change to this top-level project directory on both hosts. On ws1, run 
    ant server
  6. On ws2 (and possibly other workstations), run 
    ant -Dserverhost=ip1 client
    Now you should see some interaction between the client(s) and the server.
